## Notes on Field Values

| Field | Format / Valid Values |
|---|---|
| `registration_id` | `LR-YYYY-NNNNN` (auto-generated by API) |
| `registration_date` | `DD/MM/YYYY` |
| `year_of_registration` | 4-digit year (numeric) |
| `owner_aadhaar_last4` | Last 4 digits of Aadhaar number |
| `survey_number` | `SUR/<AREA_CODE>/<SERIAL>` |
| `land_type` | `Residential`, `Commercial`, or `Agricultural` |
| `market_value_inr` | Numeric (INR), no commas |
| `stamp_duty_inr` | Numeric (INR); typically ~5–7% of market value |
| `status` | `Active`, `Transferred`, or `Disputed` |

## How to Set Up

1. Open [Google Sheets](https://sheets.google.com) and create a **new spreadsheet**.
2. Rename **Sheet1** → `lands`
3. Click **+** at the bottom → add a new sheet → rename it → `transactions`
4. In the `lands` sheet:
   - Click cell **A1**, paste the 18 column headers
   - Paste the 10 data rows below
5. In the `transactions` sheet:
   - Click cell **A1**, paste the 6 column headers
   - Paste the 1 sample transfer row below
6. Go to **Extensions → Apps Script** and paste the contents of `Code.gs`
